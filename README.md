# blackbird-nonce-setter
A nonce setter for all devices compatible with Blackbird.

-----------------------------------------

iOS version doesn't matter anymore as you can restore to any version. If your device is compatible with blackbird. then you can set your nonce, save SHSH for unsigned versions, and downgrade.

-----------------------------------------

Support includes : 

iPhone 6s
iPhone 6s Plus
iPhone SE (1st gen, 2016)
iPhone 7
iPhone 7 Plus

iPad 5th Gen (2017)
iPad 6th Gen (2018)
iPad 7th Gen (2019)
iPad Pro (9.7 Inch)
iPad Pro (12.9 Inch 1st Gen)
iPad Pro (10.5 Inch)
iPad Pro (12.9 Inch 2nd Gen)

iPod Touch 7th Gen (2019)

-----------------------------------------

I will expand support when A8(X) is supported with turdus merula
